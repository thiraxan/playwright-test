# QA Assignment - SauceDemo Testing

## 📌 Project Overview

This project contains manual and automated tests for SauceDemo e-commerce website.

## 🚀 Installation

```bash
npm install
npx playwright install
```

## ▶️ Run Tests

```bash
npx playwright test
```

## ✅ Test Coverage

* Login (valid, invalid, empty)
* Cart (add, remove)
* Checkout flow

## 🧠 Notes

* Focused on core functionality
* Used Playwright for automation
* Structured test for readability
